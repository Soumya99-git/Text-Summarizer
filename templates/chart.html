Text Summarization


What is the purpose:
We take a text file as input(uptill now only .txt files are working)
Read text from it. Then we split the text according to the sentences
Calculate the importance of every sentence by analysing which sentence has more of the important words in it.
We then select 30% of the most important text from it.By this we summarize the contents of the text file.
A new text file is downloaded at the end
